VeriGuard Test Workbench

이곳은 네이버 뉴스, 유튜브, X(트위터), 페이스북 등의 소셜 피드 레이아웃을 정교하게 모사한 테스트 페이지입니다. 크롬 확장 프로그램(VeriGuard)이 활성화되어 있을 때, 아래 피드 내용 중 의심스러운 가짜뉴스나 딥페이크 요소가 실시간으로 분석되어 경고 배지와 테두리 하이라이트가 심어집니다.

  1. 개발자 모드 확장 프로그램 등록: 크롬 주소창에 chrome://extensions/를 입력한 뒤 '개발자 모드'를 활성화하고, Load unpacked를 클릭해 extension 폴더를 선택합니다.
  2. API 서버 연동 (선택): node mock-server/server.js를 실행하여 API 서버를 켜면 백엔드 분석 결과가 실시간으로 반영됩니다. (서버가 꺼진 경우 확장 프로그램 내부의 로컬 Heuristic 매처로 동작합니다.)
  3. 동적 포스팅 시연: 아래 제어 패널에서 가짜뉴스나 딥페이크 키워드를 활용해 새 게시글을 추가해 보세요. MutationObserver가 신규 삽입을 즉시 가로채 실시간 감지 프로세스를 수행합니다!

🛠️ 동적 콘텐츠 테스터

원하는 가짜뉴스나 딥페이크 이벤트를 클릭하면 즉시 아래 각 소셜 플랫폼 피드에 삽입되며, 확장 프로그램의 실시간 감지 기능을 확인할 수 있습니다.
